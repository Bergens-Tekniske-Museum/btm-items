# Stasjonært kompressoranlegg

* **Gjenstand:** Stasjonært kompressoranlegg
* **Produsent:** *Ukjent*
* **Type:** Kompressoranlegg med elektrisk drift med reim til kompressoren.

### Historikk og bruk
I daglig bruk fra ca. 1920 til 1965 av Bergen kommune, Bergen veivesen ved verksted og smie i Fløenbakken.

### Tekniske spesifikasjoner & komponenter

| - | - |

| **Fabrikant (elektrisk):** | Per Kure |
| **Kompressor:** | Kompressor med to sylindre |
| **Lufttank:** | Ja |
